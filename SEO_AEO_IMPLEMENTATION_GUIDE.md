## Support & Troubleshooting

### Common Issues & Solutions:

**Issue:** Schema markup not appearing in search results
- **Solution:** Wait 2-4 weeks for re-crawling; check Google Search Console for errors

**Issue:** FAQ not appearing in Featured Snippets
- **Solution:** Ensure FAQPage schema is properly formatted; query must match FAQ content

**Issue:** Low AI search mentions
- **Solution:** Increase online presence; build backlinks; maintain content freshness

**Issue:** Core Web Vitals issues
- **Solution:** Optimize images; minimize CSS/JS; use efficient fonts; enable caching
